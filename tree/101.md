## [101. 对称二叉树](https://leetcode.cn/problems/symmetric-tree)

"镜像"判断左右子树 对称节点是否相等

``` go
/**
 * Definition for a binary tree node.
 * type TreeNode struct {
 *     Val int
 *     Left *TreeNode
 *     Right *TreeNode
 * }
 */
func isSymmetric(root *TreeNode) bool {
    stack := []*TreeNode{ root.Left, root.Right }
    for len(stack) > 0 {
        l := len(stack)
        n1, n2 := stack[l-1], stack[l-2]
        // 注意这里需要先把数据弹出栈
        stack = stack[:l-2]

        if n1 == nil && n2 == nil {
            continue
        }
        if n1 == nil || n2 == nil || n1.Val != n2.Val {
            return false
        }
        stack = append(stack, n1.Left, n2.Right, n1.Right, n2.Left)
    }
    return true
}
```

``` python
# Definition for a binary tree node.
# class TreeNode:
#     def __init__(self, val=0, left=None, right=None):
#         self.val = val
#         self.left = left
#         self.right = right
class Solution:
    def isSymmetric(self, root: Optional[TreeNode]) -> bool:
        stack = [(root.left, root.right)]
        while len(stack) > 0:
            n1, n2 = stack.pop()
            if n1 is None and n2 is None:
                continue
            if n1 is None or n2 is None or n1.val != n2.val:
                return False
            stack.extend([(n1.left, n2.right), (n2.left, n1.right)])
        return True
```

时间复杂度O(n)：n 为二叉树的节点个数
空间复杂度O(n): 使用切片/列表模拟栈
